Farm Co-op Weekly Harvest List

Project Overview

AppSheet application that helps farm co-ops manage their weekly harvest lists and coordinate with local markets. Used by 4 co-ops managing approximately 50 farms.

Key Features

  • Harvest Planning: Weekly harvest list management with quantity tracking
  • Market Coordination: Integration with local markets for order fulfillment
  • Multi-Farm Support: Handles multiple farms within each co-op
  • Real-time Updates: Live data synchronization across all users

Technology Stack

  • Platform: Google AppSheet (No-Code)
  • Backend: Google Sheets with Google Apps Script automation
  • Integrations: Shopify API for market ordering
  • Authentication: Google Workspace integration

Impact

This solution streamlined the harvest coordination process for multiple farm co-ops, reducing administrative overhead and improving communication between farms and markets. The system handles weekly planning for approximately 50 farms across 4 different co-operatives.
